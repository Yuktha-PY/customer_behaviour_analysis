# Data Analytics Project

## Overview
This project demonstrates a complete data analytics workflow, starting from data loading and cleaning to visualization and reporting. The project uses Python for data analysis, SQL for querying the database, Power BI for interactive dashboards, and Gamma for creating a professional presentation.

## Dataset
The dataset contains business-related data used for analysis. It was loaded into Python for preprocessing, cleaned to improve data quality, and then imported into a SQL database for further analysis.

## Tools & Technologies
* **Python** – Data loading, cleaning, and Exploratory Data Analysis (EDA)
* **Pandas, NumPy, Matplotlib, Seaborn** – Data manipulation and visualization
* **PostgreSQL / MySQL / SQL Server** – SQL queries and database management
* **Power BI** – Interactive dashboard creation
* **Gamma** – Presentation (PPT) creation
* **Microsoft Word / PDF** – Project report documentation

## Project Workflow
### 1. Data Loading
* Imported the dataset into Python.
* Examined the dataset structure and data types.

### 2. Exploratory Data Analysis (EDA)
* Analyzed data distributions and patterns.
* Identified missing values and outliers.
* Created visualizations to understand trends.
  
### 3. Data Cleaning
* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data formats.
* Prepared the dataset for analysis.

### 4. SQL Analysis
* Imported the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
* Wrote SQL queries to:

  * Filter and sort data
  * Aggregate results using `GROUP BY`
  * Perform joins
  * Use window functions
  * Generate business insights

### 5. Dashboard Development
* Connected Power BI to the processed data.
* Created interactive dashboards with charts, KPIs, filters, and slicers.
* Designed visuals to highlight key business metrics.

### 6. Reporting & Presentation
* Documented the project methodology, analysis, and findings in a report.
* Created a presentation using Gamma to summarize the project and results.

## Dashboard
The Power BI dashboard provides:

* Key Performance Indicators (KPIs)
* Sales and revenue trends
* Category-wise analysis
* Customer insights
* Interactive filters and slicers for easy exploration

## Results
* Improved data quality through cleaning and preprocessing.
* Generated meaningful insights using SQL queries.
* Built an interactive dashboard for effective decision-making.
* Delivered a professional report and presentation summarizing the findings.



## How to Run

1. Clone or download the project.
2. Install the required Python libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Run the Python notebooks or scripts for EDA and data cleaning.
4. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
5. Execute the SQL queries provided in the project.
6. Open the Power BI file to explore the dashboard.
7. Review the project report and Gamma presentation for a summary of the analysis.
